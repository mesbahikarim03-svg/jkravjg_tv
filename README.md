# Universal Downloader (Android)

A modern Android application that downloads publicly accessible files from the internet using a single URL.

## Features
- Smart URL validation
- Automatic filename detection
- Uses Android native DownloadManager
- Supports direct file links (MP4, MP3, PDF, ZIP, JPG, PNG, APK, etc.)
- Material Design UI
- Android 13+ compatible

## Setup
1. Open project in Android Studio (Hedgehog or newer)
2. Sync Gradle
3. Run on device (API 26+)

## Note
This app supports only publicly accessible direct file URLs.
